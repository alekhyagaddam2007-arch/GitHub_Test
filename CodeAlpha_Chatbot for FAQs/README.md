# FAQ Chatbot

## Description
A simple FAQ Chatbot built using Python and Tkinter. It answers frequently asked questions related to Python, Artificial Intelligence, Machine Learning, and Data Science.

## Features
- Simple GUI
- Programming FAQs
- User-friendly interface
- Instant responses

## Technologies Used
- Python
- Tkinter

## How to Run
1. Open the project folder.
2. Run:
   python faq_chatbot.py
